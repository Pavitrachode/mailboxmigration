🚀 AI-Powered Mailbox Migration Assistant
Automating On-Prem to Azure Cloud Mailbox Migration (Node.js Version)

📌 Overview

This project is an AI-powered mailbox migration assistant built using Node.js and deployed on Replit. It is designed to simplify and automate the migration of mailboxes from on-premises Exchange to Microsoft Azure / Exchange Online.

The idea for this project came from my real-world experience performing mailbox migrations as an IT Analyst. The manual process can be complex, repetitive, and error-prone — so this application uses AI to guide the steps, identify issues, and streamline the entire workflow.


✨ Key Features
🔹 AI-generated guidance for every migration stage
🔹 Pre-migration checks for common hybrid issues
🔹 Automated task generation for mailbox transfers
🔹 Error detection & troubleshooting suggestions using LLMs
🔹 Web-based interface built on Node.js
🔹 PostgreSQL-backed storage for logs and history
🔹 Lightweight and deployable directly from Replit
